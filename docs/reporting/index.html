üìà Campaign Data

Report.AI integrates directly with the Lumina platform to fetch campaign information and automatically detect tactics.

Lumina URL Processing

The system accepts Lumina platform URLs and automatically extracts the campaign order ID for API processing.

üí° Supported URL Formats

Report.AI supports any Lumina URL containing a 24-character hex ObjectID. The system automatically strips query parameters like ?tab=line_items or ?tab=tasks.

Example Lumina URL: https://townsquarelumina.com/lumina/view/order/677430e377ff89a87ff6c62f?tab=line_items Extracted Order ID: 677430e377ff89a87ff6c62f

üìÅ File Upload System

The most efficient way to upload data. Simply drag and drop all your CSV files, and the system will automatically sort them to the correct tactics and tables.

1

Filename Pattern Recognition

Files with patterns like report-targeted-display-monthly-performance.csv are automatically routed to the correct table.

2

Header Similarity Matching

If filename patterns don't match, the system analyzes CSV headers using Jaccard similarity (‚â• 0.6 threshold).

3
